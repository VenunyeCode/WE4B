FAQ!

Pour s'inscire sur Winku, c'est simple, il faut allé sur Winku (https://www.winku.com).
  1. Cliquez sur le bouton "S'inscrire" sur la page de notification
  2. Renseigner vos informations personnelles
  3. Puis cliquer sur "S'inscrire" pour valider
  4. Voilà vous serai dirigé vers votre espace
Dans votre espace personnelle cliquez dans le menu à gauche "Profil" puis "Edition profil" sur la page qui souvre. Vous pouvez également cliquez sur votre avatar dans le coin à droite, se déroule un petit menu dans lequel vous trouverez "Edition profil". Une fois sur la page d'édition, renseigner vos informations et cliquez sur le bouton "Modifier"
